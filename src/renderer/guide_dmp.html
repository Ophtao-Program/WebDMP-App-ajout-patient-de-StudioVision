📖 Guide WebDMP Assistant

Cet outil vous permet d'ouvrir le portail Web DMP de Mon Espace Santé, de détecter automatiquement le patient ouvert dans StudioVision, et de copier facilement ses notes et documents dans le DMP.

Workflow quotidien

Étape 1 — Ouvrir le Web DMP
Cliquez sur 🌐 Ouvrir Web DMP. Une fenêtre s'ouvre sur le portail dmp.fr.
Connectez-vous avec votre carte CPS ou votre identifiant e-CPS (le champ est pré-rempli si vous l'avez mémorisé).
Étape 2 — Détecter le patient
Ouvrez le dossier du patient dans StudioVision, puis cliquez sur ⟳ Détecter le patient actif dans StudioVision.
Le nom, la date de naissance et le NIR s'affichent automatiquement.
Étape 3 — Consulter les documents
L'onglet Documents liste les derniers fichiers PDF/images du patient (comptes rendus, imagerie…).
Cliquez sur 👁 pour ouvrir un fichier dans son lecteur habituel (Acrobat, Aperçu…).
Cochez les documents que vous souhaitez déposer dans le DMP.
Étape 4 — Copier des notes
L'onglet Notes liste les dernières consultations/observations du patient.
Cliquez sur 📋 Copier tout pour copier le texte dans le presse-papier.
Dans la fenêtre Web DMP, collez le texte (Ctrl+V) dans le champ correspondant.

Mémoriser votre identifiant e-CPS

Saisissez votre identifiant e-CPS dans le champ prévu, puis cliquez sur 💾 Mémoriser l'identifiant. Il sera automatiquement pré-rempli à chaque ouverture de l'application. Cet identifiant est stocké localement sur votre poste, jamais transmis.

Prérequis techniques

  • StudioVision doit être ouvert avec un dossier patient visible pour la détection automatique.
  • pywin32 doit être installé : pip install pywin32
  • pyodbc + pilote Access 64 bits pour l'accès à PUBLIC.MDB.
  • La configuration des chemins (FICHIER_MDB, DEST_PHOTOS) doit être adaptée dans python/dmp_connector.py.
💡 Conseil : La fenêtre Web DMP conserve votre session entre les utilisations (cookie persistant). Vous n'avez à vous connecter qu'une seule fois par journée de travail.
